# SPEC.md — 기능 명세

## MVP 핵심 기능 (이것만 있으면 서비스 가능)

### 1. 온보딩 — GitHub 연동 + 설문

**목적**: GitHub 데이터만으론 "왜 기여하려는지", "실제 수준이 어느 정도인지" 알 수 없음.
둘을 합산해서 유저 프로필 생성.

**GitHub에서 자동 수집하는 것**
- 주요 사용 언어 (레포별 언어 비율 합산)
- 커밋 수 (최근 6개월)
- 기여한 레포 평균 스타 수 (규모 파악용)

**설문에서 물어보는 것**

| 항목 | 선택지 |
|---|---|
| 개발 경력 | 0~1년 / 1~3년 / 3년 이상 |
| 기여 목적 | 포트폴리오 / 실력 향상 / 커뮤니티 참여 |
| 선호 기여 방식 | 코드 / 문서·번역 / 둘 다 |
| 주당 투자 가능 시간 | 2시간 이하 / ~5시간 / 10시간 이상 |
| 영어 이슈 소통 가능 여부 | 가능 / 불가능 |

**온보딩 완료 후**: `user_profiles.onboarding_done = true` 설정, 대시보드로 리다이렉트

---

### 2. 이슈 매칭 (대시보드)

**핵심 로직**: GitHub GraphQL로 이슈 수집 → 규칙 기반 스코어링 → 유저 프로필과 매칭

**이슈 수집 조건**
- `is:open is:issue` — 열려있는 이슈만
- `label:good-first-issue OR label:help-wanted`
- 레포 활성도 점수 40점 이상인 것만 (방치된 레포 제외)

**표시 정보**
- 이슈 제목 / 레포명 / 언어
- 기여 방식 태그 (doc / bug / feat / test)
- 난이도 (입문 / 초급 / 초중급 / 중급 이상)
- 레포 활성도 점수
- 경쟁도 (코멘트 수 / 연결된 PR 유무)
- 마지막 활동 시각

**필터**
- 기여 방식
- 난이도
- 언어

---

### 3. 레포 활성도 점수

**목적**: 기여해도 메인테이너가 반응 안 하는 레포를 사전에 걸러줌.

**점수 계산 (0~100)**

| 지표 | 가중치 | 계산 방식 |
|---|---|---|
| 최근 커밋 | 30점 | 30일 이내 → 30점, 90일 이내 → 20점, 180일 이내 → 10점, 초과 → 0점 |
| PR 평균 응답 속도 | 30점 | 3일 이내 → 30점, 7일 이내 → 20점, 14일 이내 → 10점, 초과 → 0점 |
| PR 머지율 | 25점 | 80%+ → 25점, 60%+ → 15점, 40%+ → 5점, 미만 → 0점 |
| 이슈 응답 여부 | 15점 | 최근 이슈에 메인테이너 코멘트 있음 → 15점 |

**40점 미만**: 기본 노출 제외 (유저가 필터 해제 가능)
**캐시**: 1시간 동안 `repo_health_cache` 테이블에 저장 후 재사용

---

### 4. 이슈 경쟁도 표시

| 상태 | 기준 |
|---|---|
| 🟢 여유 | 코멘트 0~1개, 연결된 PR 없음 |
| 🟡 경쟁 중 | 코멘트 2개 이상 또는 관련 코멘트 있음 |
| 🔴 진행 중 | 연결된 PR이 이미 존재함 |

---

### 5. 북마크 + PR 상태 추적

- 이슈 북마크 저장
- 상태 수동 변경: `saved → in_progress → pr_open → merged / abandoned`
- PR URL 직접 입력 시 연결

## 추가 기능 (MVP 이후)

| 기능 | 설명 | 난이도 |
|---|---|---|
| 코드 맥락 가이드 | 이슈 선택 시 관련 파일·함수 위치 + 수정 방향 제안 | 높음 |
| 기여 타임라인 | 북마크 → PR → 머지까지 히스토리 시각화 | 중간 |
| 머지 뱃지 + 커리어 페이지 | PR 머지 시 자동 뱃지, 이력서용 문장 생성, 공유 링크 | 중간 |
| 주간 이슈 추천 메일 | Vercel Cron으로 매주 새 이슈 발굴 후 이메일 발송 | 낮음 |
| 커뮤니티 후기 | 같은 레포 기여자 경험 공유, 난이도 재평가 | 높음 |
| 프로필 재보정 | 기여 이력 쌓이면 매칭 기준 자동 상향 | 중간 |

## 유저 플로우

```
GitHub 로그인
    ↓
온보딩 설문 (처음 방문 시)
    ↓
프로필 생성 (GitHub 분석 + 설문 합산)
    ↓
대시보드 — 이슈 매칭 결과
    ↓
이슈 북마크
    ↓
PR 상태 추적
```
